#  FastFood Web Application

A full-stack web application built using ASP.NET MVC and SQL Server that simulates a fast food ordering system with two roles: Admin and User.

---

##  Features

### Admin
- Add new food items to the menu
- Edit and update existing items
- Delete items from the menu
- Manage and review the menu

###  User
- Browse available food menu
- Add items to cart
- Update cart (increase/decrease quantity)
- Remove items from cart

---

##  Technologies Used

- ASP.NET MVC (.NET Framework / .NET Core)
- C#
- SQL Server
- HTML5, CSS, JavaScript

---

##  Project Structure

- Models → Handles data and database logic
- Views → UI and pages
- Controllers → Application logic
- Database → SQL Server with relational tables

---

##  System Roles

| Role  | Permissions |
|-------|------------|
| Admin | Manage menu (Add, Edit, Delete) |
| User  | View menu, manage cart |

---

##  How It Works

1. Admin logs in and manages the menu.
2. Users browse the menu and add items to their cart.
3. Users can update or remove items from the cart before checkout.
